# Código de Conduta

Este projeto adota o **Contributor Covenant** como base.

## Nosso compromisso

Nos comprometemos a tornar a participação nesta comunidade uma experiência livre de assédio para todos, independentemente de idade, tamanho corporal, deficiência, etnia, características sexuais, identidade e expressão de gênero, nível de experiência, educação, status socioeconômico, nacionalidade, aparência pessoal, raça, religião ou identidade sexual.

## Nossos padrões

Exemplos de comportamentos que contribuem para um ambiente positivo:

- Uso de linguagem acolhedora e inclusiva
- Respeito por pontos de vista e experiências diferentes
- Abertura a feedback construtivo
- Foco no que é melhor para a comunidade

Exemplos de comportamentos inaceitáveis:

- Assédio, insultos e ataques pessoais
- Publicação de informações privadas sem permissão
- Qualquer conduta inadequada em contexto profissional

## Aplicação

Para reportar incidentes, abra uma issue (se apropriado) ou entre em contato com o mantenedor do repositório.
